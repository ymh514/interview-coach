# Prompt — Cheat Sheet(工程師版)

用法:面試前貼進 Claude,它會讀 repo 全部 context 客製。只要條列、可掃視。

---

幫我準備這場面試的 cheat sheet,參考 `corpus/resume.md`、`corpus/`、這家 `companies/<x>/` 的所有 context,針對「我、這家公司、這個職缺、這位面試官」客製。只用條列、可掃視格式。

- **英文自我介紹**(30 秒精簡版,結尾帶這家的鉤子)
- **這場的策略重點**:面試官想驗證什麼?我怎麼贏這場?
- **根據前幾場 feedback-log 要修正的點**
- **要主動帶出的技術故事**(對應此團隊技術棧,從 story-bank 挑)
- **各故事的關鍵數字**(★ key numbers)
- **中文 Q&A 預備**:薪資(見 private/preferences.md)、到職通知期、為什麼離開、為什麼選這家
- **要反問面試官的問題**(顯示我懂他們的問題領域)
- **標清楚哪段用英文、哪段用中文**
